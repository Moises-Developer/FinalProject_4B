#DOSSIER DES COMPOSANTS

Cet dossier il y a des composants que se va utiliser
pour faire le rendu de la page donc quand on fais 
une partie de la page que on a besoin d'utiliser pour
beaucoup de code source, il faut que faire une fonction
avec le rendu de la partie de la page.

Les composants ce sont très important pour on peut faire
une web page très organisé et on peut faire les choses plus
facilement.
